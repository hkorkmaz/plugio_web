Metric Cards & KPI Dashboard Documentation

Plugio Metric Cards lets you build a dashboard of focused KPI tiles backed by JQL. Combine multiple card types in a single gadget to surface counts, progress against targets, trends, ratios, top issues, and breakdowns — all on one Jira dashboard.

Getting Started

Add the gadget to your Jira Dashboard in 4 steps:

  1. 1 On your Jira Dashboard, click Add Gadget.
  2. 2 Find "Metric Cards & KPI Dashboard" and click Add.
  3. 3 Click Configure on the gadget to open the editor.
  4. 4 Add one or more metrics, pick a card type for each, and set the comparison period.

Card Types

Each card represents a single metric. Pick the card type that matches the question you want to answer:

1. Basic

A single value with optional period-over-period delta. Use it for headline numbers like "Open Bugs" or "Resolved This Week".

2. Progress

Current value against a target, rendered as a progress bar. Targets can be a fixed number or dynamically derived from a second JQL query (e.g. resolved vs. total in scope).

3. Trend

A sparkline of the metric across buckets in your selected period (day / week / month / quarter). Great for spotting direction at a glance.

4. Ratio

Two JQL queries combined into a percentage or rate — e.g. bug ratio, completion rate, escaped defects.

5. Distribution

Breaks the metric down by a categorical field (status, priority, assignee, custom field) and renders it as a compact bar or pie chart inside the card.

6. List

Top-N issues from the configured JQL, ordered by your chosen field. Useful for "Top open blockers" or "Most recently updated" tiles.

Data & Periods

Every card is backed by a project selection and an optional JQL filter. The gadget runs each query using the viewer's permissions, so users only see issues they have access to.

Comparison Periods

Set a single period at the gadget level — Basic, Progress, and Ratio cards will automatically compute a previous-period delta against it. Trend cards use the period to define the X axis buckets.

Aggregations

For numeric fields you can aggregate as count, sum, min, or max. The default is issue count.

Layout & Theming

Cards arrange themselves into a responsive grid that fits the gadget container — drag to resize the gadget on your Jira dashboard and the cards reflow automatically.

Per-card color accents let you signal severity or category. The gadget honors Jira's light and dark themes out of the box.

World Map — Country & Region Analytics

Plugio World Map lets you visualize monday.com board data on an interactive map — colored by country or region. See where your customers, deals, or projects are concentrated at a glance, with three chart types, 10 built-in map regions, timeline filtering, and dynamic conditional coloring.

Getting Started

Follow these steps to add the widget to a monday.com dashboard:

  1. 1 Open any monday.com dashboard and click Add Widget.
  2. 2 Search for "Plugio World Map" and add it to your dashboard.
  3. 3 Click Settings on the widget to open the configuration panel.
  4. 4 In the Data tab, select your board and the column that contains location or country information.
  5. 5 Choose your operation (Count, Sum, Average, etc.) and optionally select a numeric column to aggregate.
  6. 6 Click Save to apply your configuration.

Visualization Modes

Switch between three visualization modes in the Map Style tab without changing your data setup.

Choropleth

Countries or regions are filled with a color gradient based on their value — darker means higher, lighter means lower. This is the default mode and works best for showing distribution across a large number of regions.

Bubble

A circle is drawn at the center of each region, sized proportionally to its value. Larger bubbles immediately draw attention to the highest-value regions. Works well when you want to compare magnitudes rather than distributions.

Pie Chart

A mini pie chart is rendered on each region, breaking down the data by a Grouping Column you select (such as a Status or Dropdown column). Each slice represents a category. Use this mode to show composition alongside geographic distribution.

To use Pie Chart mode, select a Status or Dropdown column as the Grouping Column in the Map Style tab.

For country-level maps (US, UK, Germany, etc.), you will need to select a Region Column — a column in your board that contains state, province, or region names for that country. The widget supports full names, common abbreviations, and alternative spellings.

Data Configuration

All data settings are in the Data tab of the configuration panel.

Location Column

Select the column that identifies the geographic location of each item. The widget supports:

  • Location column: Automatically extracts the country from the address.
  • Country column: Uses the built-in monday.com country picker.
  • Text or Dropdown column: Accepts country names (e.g. "Germany"), ISO-2 codes (e.g. "DE"), or common abbreviations.

Operation

Choose how the widget aggregates item values per region:

  • Count: Number of items per region. No numeric column needed.
  • Sum: Total of a numeric column per region.
  • Average: Mean value of a numeric column per region.
  • Minimum / Maximum: The lowest or highest value in a numeric column per region.

Measure Column

When using Sum, Average, Min, or Max, select the numeric column to aggregate. Supports Numbers and Formula column types.

Conditional Coloring

By default, the map uses a single-color gradient scaled to your data range. You can switch to Conditional mode in the Coloring tab to define custom color ranges based on specific thresholds.

Setting Up Color Ranges

Each color range has a start value, an end value, and a color. Ranges chain automatically — the end of one range becomes the start of the next. Click Add Range to add more.

Dynamic Thresholds with Variables

Threshold values can be static numbers or dynamic values pulled from your board using Variables. For example, set the upper threshold to a variable that calculates the average deal value across all items — the color ranges will automatically adjust as your data changes.

Type @ in any threshold field to reference a variable, or click the variable icon to create a new one.

Timeline & Playback

The Timeline tab lets you connect a Date column to the map, enabling time-based filtering and animation.

Date Filter

Adds an interactive date range slider below the map. Drag the handles to filter which items are included in the map based on their date column value. Useful for exploring how your geographic data looked during a specific period.

Playback

Animates the map through time automatically. Configure the Period to control the step size:

  • Daily, Weekly, Monthly, Quarterly, Yearly
  • Month of Year — compare the same month across all years
  • Day of Week — compare activity by weekday
  • Hour of Day — for time-stamped data

Enable Cumulative to show all data up to the current playback position, rather than only the data within the current period.

Variables

Variables are reusable data queries that resolve to a single number from your board. They can be used as dynamic threshold values in Conditional Coloring.

Creating a Variable

Open the Variables manager from the Coloring tab, or type @ in any threshold field and select Create variable.

Variable Types

  • Board Query: Select a board, a column, and an aggregation (Sum, Average, Count, Min, Max) with optional filters. The variable resolves to that computed value in real time.
  • Formula: Combine multiple variables using a mathematical expression, for example (revenue / target) * 100.

Filters

Each board query variable supports column-level filters — limit the query to specific groups, statuses, assignees, or any other column type. This lets you define precise thresholds such as "average deal value for Won deals only."

FAQ

Does my data leave monday.com?

No. The widget operates entirely within the monday.com platform using the monday.com Apps SDK. All data fetching and processing happens within your monday.com instance. We never store or transmit your board data.

Which column types are supported for location?

Location, Country, Text, Long Text, Dropdown, Status, and Name columns are all supported. The widget normalizes country names and common abbreviations automatically.

My country isn't being matched on the map. What should I check?

Ensure the column values match either a country name in English (e.g. "Germany"), an ISO-2 code (e.g. "DE"), or a common alias. If using a Location column, make sure the location resolves to a valid country in monday.com.

Can I use data from a board that isn't connected to the dashboard?

The widget loads all boards available to your account. You can select any board regardless of whether it is connected to the current dashboard.

How many items can the widget handle?

The widget uses cursor-based pagination and fetches all items from the selected board, regardless of size. Performance may vary for very large boards (10,000+ items).

Can I use the Pie Chart mode without a numeric column?

Yes. In Pie Chart mode, select a Grouping Column (Status or Dropdown). The widget will count items per category per region. No numeric column is required.
